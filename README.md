# YouTube Ad Accelerator

YouTubeの動画広告を自動で検知し、高速再生と自動ミュート、自動スキップを行うことで、快適な動画視聴をサポートするChrome拡張機能（Manifest V3対応）です。

## 主な機能

*   **広告の自動検知と高速再生**: 広告が開始されると自動的に検知し、指定された速度（2倍〜最大16倍速、デフォルト5倍）で早送りします。
*   **自動消音（ミュート）**: 広告再生中は自動的に音声をミュートにし、不快な広告音をカットします。広告終了後は自動で元の音量・ミュート状態に復元します。
*   **自動スキップ**: スキップボタンが表示されたら即座に自動クリックします。また、広告のシークバーを自動的に末尾まで進めることで、実質的に瞬時に広告を終了させます。
*   **ダッシュボード（統計情報）**:
    *   スキップした広告の累計数
    *   節約できた時間の累計秒数/分数
*   **シンプルな設定UI**: ポップアップ画面から拡張機能のオン/オフ、再生速度の調整（2x〜16x）、自動ミュート・自動スキップの個別設定が可能です。

## インストール方法（開発者モードでの導入）

本拡張機能はChromeウェブストアを介さずに、ローカルで読み込んで使用することができます。

1.  本リポジトリをダウンロード（ZIP解凍）またはクローンします。
2.  Google Chromeを開き、アドレスバーに `chrome://extensions/` と入力してアクセスします。
3.  画面右上にある **「デベロッパー モード」** のトグルスイッチをオンにします。
4.  画面左上に表示される **「パッケージ化されていない拡張機能を読み込む」** ボタンをクリックします。
5.  ダイアログが表示されるので、解凍したフォルダ内の `ad-accelerator` ディレクトリ（`manifest.json` が含まれているフォルダ）を選択します。
6.  拡張機能の一覧に「YouTube Ad Accelerator」が表示されればインストール完了です。

## 使い方

1.  YouTube（`https://www.youtube.com/`）にアクセスして動画を再生します。
2.  広告が表示されると、拡張機能が自動的に作動して広告を高速再生＆ミュート化し、スキップ可能な状態になり次第自動でスキップします。
3.  Chromeのツールバーにあるパズルピースのアイコンから「YouTube Ad Accelerator」をピン留めすることで、いつでもポップアップ画面から設定の変更や統計情報の確認を行えます。
4.  統計情報はリアルタイムで更新され、「統計をリセット」ボタンをクリックすることでいつでも初期化できます。

## 技術仕様

*   **マニフェストバージョン**: Manifest V3
*   **技術スタック**: HTML5, CSS3, JavaScript (ES6+)
*   **堅牢な検知システム**: YouTubeのDOM構造の変化を `MutationObserver` で常に監視し、Shadow DOMの内部にあるスキップボタンも再帰的に検知（Deep Query Selector）して確実にクリックを実行します。また、YouTube側のスクリプトによる再生速度・音量のリセット処理をイベントの同期制御によってインターセプト（遮断）する仕組みを備えています。
